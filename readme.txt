== Description ==
A plugin for managing & displaying adbirt ads on a wordpress site. See https://adbirt.com/privacy & https://adbirt.com/terms for TAC/privacy policy.

Feature list:

* Display Adbirt ads on your site via shortcode the `[adbirt_ads_display]` shortcode.
* Display Adbirt ads on your site via the **Adbirt widget**.
* Display either a single ad, or a rotating banner showind all ads, or just ads for the specified category.
* Manage Adbirt Ads on your site.
* Add custom categories for ads.

== Installation ==
1. Install the plugin from the wordpress.org plugin directory directly, or download the zip file and upload \"/wp-content/plugins/\" directory.
2. Activate the plugin through the *\"Plugins\"* menu in WordPress.
